# Create React Appのはじめかた
> 公式 : [create-react-app](https://github.com/facebookincubator/create-react-app)  

## インストールと起動
```bash
# カレントディレクトリ : ~/.
npm i -g create-react-app
create-react-app polyreact
cd polyreact/
npm start
```

→ http://localhost:3000 を開くと、 `polyreact` の内容が表示されます。
